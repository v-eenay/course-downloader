# Course Downloader

Local desktop application for downloading Coursera course materials.

## Features

- Downloads videos, assignments, notes, and other course resources in a structured folder layout.
- Supports full specialization inputs and expands them into course folders under a specialization root.
- Accepts higher video-quality targets such as 1080p, 1440p, 2160p, or the best available source.
- Supports automatic browser authentication across the shared browser list used by both the GUI and CLI.

## Usage Notes

- Sign in to coursera.org in the browser you select before starting a download.
- You can provide either a course URL/slug or a specialization URL/slug.
- Resume uses the same input and destination folder to continue an interrupted download.
